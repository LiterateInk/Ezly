## Installation

### Maven

```xml
<dependency>
  <groupId>ink.literate</groupId>
  <artifactId>ezly</artifactId>
  <version>0.0.0</version>
</dependency>
```

### Gradle (Kotlin)

```kotlin
implementation("ink.literate:ezly:0.0.0")
```

### Gradle

```groovy
implementation 'ink.literate:ezly:0.0.0'
```
