## Installation

Use your favorite package manager to install this library from NPM.

```bash
# pnpm
pnpm add ezly

# Yarn
yarn add ezly

# npm
npm add ezly

# Bun
bun add ezly
```

## Documentation

You can find documentation with guides at [docs.literate.ink/ezly](https://docs.literate.ink/ezly).
